# Roadmap

---

## v1 — Core Intelligence

**Goal:** Working cross-vertical agent that detects food mode and surfaces proactive insights.

- [ ] Swiggy OAuth integration
- [ ] Cross-vertical data fetch from all 3 MCP verticals
- [ ] Food mode detection engine (Cooking / Ordering / Social)
- [ ] 4 proactive insight card types
- [ ] Natural language chat over full order history
- [ ] Cross-vertical spend dashboard
- [ ] Basic food personality label ("Weekend Chef, Weekday Orderer")

---

## v2 — Behavioral Depth

**Goal:** Richer insights that make users feel genuinely understood.

- [ ] Behavioral drift detection ("You used to cook 3x/week — you stopped 3 weeks ago")
- [ ] Cuisine evolution timeline — how your taste has changed over months
- [ ] Instamart basket intelligence ("You buy paneer but always order paneer dishes — try cooking them?")
- [ ] Shareable food personality card (Spotify Wrapped format)
- [ ] Social dining patterns — who you go out with, when, where

---

## v3 — Predictive Layer

**Goal:** Move from reactive insights to proactive suggestions.

- [ ] Mode transition prediction ("You're about to shift to Ordering Mode — here's a quick Instamart restock suggestion")
- [ ] Budget forecasting across all 3 verticals
- [ ] Seasonal behavior patterns
- [ ] Personalized restaurant/grocery suggestions grounded in behavioral history
